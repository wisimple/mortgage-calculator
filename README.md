# Mortgage Calculator

Live demo <a href="https://calculate-mortgage.netlify.app/" target="_blank">https://calculate-mortgage.netlify.app</a>

### Installation

1. Clone or download the project
2. Go to project directory
3. `$ npm install`
4. `$ npm start`
